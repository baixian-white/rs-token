## Compile (Windows / Linux / macOS)

```bash
pdflatex rs_token
bibtex   rs_token
pdflatex rs_token
pdflatex rs_token
```

The four-pass sequence is required because: pass 1 emits citation keys, BibTeX builds the bibliography, then two more pdflatex passes resolve cross-references and bibliography labels.

## Compatibility Notes

- The `IEEEtran` class (v1.8b or later) ships with most TeX Live distributions; if your local distribution does not include it, install via `tlmgr install IEEEtran`.
- Unicode characters (Chinese, em-dashes, ×) may need `\usepackage[utf8]{inputenc}` (already enabled).
- The architecture figure file uses a typo-preserving filename `fig_method_aech_image2pro.png` (matches the existing artifact); rename to `fig_method_arch.png` consistently across `\includegraphics` and the file system if cleaner naming is preferred.
